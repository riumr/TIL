#### 마크다운 문법 실습 : Typora

1. Heading

   - 제목/소제목

2. List (목록)

   - 순서가 있는 리스트 : 1. 2. 3. ....

3. Fence code block
   - backtick 3개 (` ``` `) 사용
   - syntax highlight

```python
print("hello world") # python code
```

4. Inline code block : ` `` `(backtick 두 개) 사용
5. Link : `[문자열](url)` 사용
