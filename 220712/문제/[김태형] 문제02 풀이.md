## 문제

> 문자열 word의 길이를 출력하는 코드를 for문으로 작성하시오.

**len() 함수 사용 금지**
> 

## 접근 방법

변수에 문자열의 길이만큼 1을 더해준다.

## 코드

```python
# for
count=0
for i in word:
	count+=1
print(count)
```
