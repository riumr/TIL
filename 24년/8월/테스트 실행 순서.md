## 테스트 실행 순서

### 테스트 환경 설정

테스트 계획서에 명시된 HW, SW 환경을 따름

### 테스트 케이스 실행

`자동화 도구`를 사용할 경우 테스트 자동화 스크립트를 실행

### 테스트 케이스 결과 분석

실행결과에서 에러가 발생하면 결함일 경우 `결함 관리 대장`에 기록

### 테스트 자산 개선

테스트를 수행하면서 변경된 테스트 케이스, 테스트 요구사항, 테스트 데이터 등을 업데이트

### 테스트 결과 작성

`테스트 케이스 명세서`에 해당 테스트 실행 결과 작성

### 테스트 실행 결과 검토

작성된 테스트 결과 및 결함 조치 내역을 취합해 프로젝트 관리자와 검토 및 협의
